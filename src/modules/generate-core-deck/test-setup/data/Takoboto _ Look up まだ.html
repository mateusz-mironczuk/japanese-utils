まだ, 未だ, いまだ
Adverb, Usually in kana
1. still, as yet, only
with verb in the negative
2. (not) yet
Only まだ
3. more, (more) still
4. at least, comparatively, relatively
NA-adjective
5. unfinished, incomplete, not yet done
私はまだその問題が解けない。
